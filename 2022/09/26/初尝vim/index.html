vim初尝

最近了解了一下vim编辑器，挺感兴趣就写篇博客总计输出一下 这里的内容只是简单入门

4种模式

首先vim具有4种模式
普通模式
输入模式
命令模式
可视模式

模式间的转换

普通模式下
i 光标定位到当前字母的前面并进入输入模式
大写i 光标定位到当前行的前面并进入输入模式
a 光标定位到当前字母的后面并进入输入模式
大写a 光标定位到当前行的后面并进入输入模式
esc 退出输入模式 返回普通模式
v 进入可视模式 用于选中内容

普通模式

普通模式下可以进行移动光标 编辑（剪切、复制、粘贴）等操作
普通模式无法输入

move cursor 移动光标

普通模式下
hjkl => 左下上右
数字键+hjkl可以一次移动多格
w 跳转到下一个单词的开头 相当于ctrl right
b 跳转到上一个单词的开头 相当于ctrl left
ctrl u => pageUp
ctrl d => pageDown
双击 g 跳转到文档开头 相当于home
大写g 跳转到文档末尾 相当于end
f 相当于find f+任意一个字母 将光标移动到离你最近的字母上

edit

normal mode
y => copy
yy => copy current line
yaw => copy all word
y4j => copy next 4lines
p => paste
c => change
cc => change current line
d => delete
dd => delete current line
daw => delete all word
d4k => delte up 4lines
u => undo ctrl z

命令模式

: 进入命令模式
q quit 退出文件
w write 覆写（保存）这个文件
qw 保存并退出
esc 退出命令模式

可视模式

v 进入可视模式
进入可视模式 移动光标选中内容

plugin 插件

vim有丰富的插件
比如 目录树插件 nerdTree
coc.nvim 自动补全插件
主题插件等等 可自行探索
